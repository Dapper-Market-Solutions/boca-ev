## 2. General image specifications

Apply these to **every** cart photo unless otherwise noted:

| Spec | Requirement |
|---|---|
| **Format** | JPG (preferred) or high-quality PNG |
| **Resolution** | **Exactly 1600 × 1000 pixels** (16:10 landscape) |
| **Orientation** | Landscape (horizontal) |
| **Background** | Clean white, seamless studio, or consistent neutral |
| **Angle** | 3/4 front view (front-left corner visible), same angle across all colors of the same model |
| **Lighting** | Even, no harsh shadows on body; minor reflections OK |
| **Framing** | Cart centered, leaving ~10% margin on all sides |
| **File size** | Under 5 MB per photo (we'll compress for web) |
| **Color accuracy** | Represent the true paint color — calibrated monitor export preferred |
| **File extension** | Must be `.jpg` (or `.png` if background transparency needed) |

> **Why the same angle matters:** When a visitor clicks different color chips on a model card, the image swaps. If each color is shot at a different angle, the cart appears to "jump around." Consistency = professional feel.

## 14. Dealer locator map — decision needed

The site's primary goal is **attracting new dealers**. However, retail consumers will also land on the site. We need your direction on whether to include a **dealer locator map** (searchable map showing existing dealer locations).

**Pros of including it:**
- Consumers can find an authorized dealer near them instead of bouncing
- Signals the brand has real distribution and is an established network
- Lets dealers show up on the brand site — a perk they can use to sell to

**Cons / concerns:**
- Existing dealers may see new dealers as **competition in their area** — could cause friction
- As the network grows, territories may overlap on the map
- Managing the map requires ongoing upkeep (dealer onboarding/offboarding)

**Please choose one:**

- [ ] **Yes — include a dealer locator map.** We'll set territory rules and share dealer data.
- [ ] **No — keep dealer info private.** Inquiring consumers will be referred by the sales team directly.
- [ ] **Partial — show a "Find a Dealer" form** that captures the visitor's ZIP and we refer them manually, without publishing the full map.

If yes or partial, please also provide:
- [ ] Current dealer list (name, address, phone, website, territory/region)
- [ ] Policy on how new dealers are added to the map
- [ ] Whether dealer logos should appear alongside their listing
